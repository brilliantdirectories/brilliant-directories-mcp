# Job Publishing Routine

Connect to my Brilliant Directories site MCP and run the `bd-skill-content` skill to autonomously create job listing posts.

Create exactly:

- COUNT: `1` job post
- PUBLISH STATUS: `post_status=1`
- AUTHOR: `user_id=1`

Pick job listings that fit the niche, carry strong local search value, and whose subject matter naturally connects to related pros, categories, and locations a sentence can link to.

Prefer listings with strong local search intent and public application info available; when a posted-date shows, favor the last 30 days, but a real on-topic listing whose page shows no posted-date is fully usable.

If an application URL is known, include this CTA immediately after the intro paragraphs:
