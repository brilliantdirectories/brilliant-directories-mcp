# Event Publishing Routine

Connect to my Brilliant Directories site MCP and run the `bd-skill-content` skill to autonomously create event posts.

Create exactly `1` fully published event post (`post_status=1`, `user_id=1`).

Pick events that match my site's focus and niche, strengthen local-SEO and topical authority, and create internal-linking opportunities to related events, locations, and members already on the site.

Prefer events with strong local search intent, recurring annual relevance, official source information available, and geographically relevant supporting content already on the site.

If a registration or official information URL is known, include this CTA immediately after the intro paragraphs:

```html
<a href="EVENT_URL" class="btn btn-secondary btn-lg vmargin" title="<descriptive>">
  Register or Learn More
</a>
```

Run fully autonomous: no clarification, no approval gates, no confirmation pauses. Analyze, pick the best opportunity, publish.

Final output: event title, location, category, internal-linking strategy used, published URL.
